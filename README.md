**Developers**

1. Add the repository
```xml
<repository>
    <id>jitpack.io</id>
    <url>https://jitpack.io</url>
</repository>
```

2. Add the dependency
```xml
<dependency>
    <groupId>com.github.luisBazanDev</groupId>
    <artifactId>KitsAPI</artifactId>
    <version>1.1</version>
</dependency>
```
